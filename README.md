# 躲貓貓 🐾

一個「潛水員戴夫」像素風的找貓小遊戲：貓咪家族躲在合照人群裡，會用各種姿勢（坐／趴／睡／站）冒出來約 1 秒又消失，趁牠出現時點到牠，全部抓到就計時過關。含慵懶木吉他背景音樂、通關旋律與地面流動波光。

## 玩法
- 趁貓探頭時點擊 → 抓到
- 4 隻貓全部抓到 → 結算時間與評級
- 右上角可切換音樂

## 執行
```bash
python3 -m http.server 8753
# 瀏覽器開 http://localhost:8753/index.html
```

## 內容物
- `index.html` — 遊戲本體（HTML/CSS/JS，無相依套件）
- `cat_<color>_<pose>.png` — 16 張貓咪像素 sprite（4 色 × 4 姿勢），由 AI 生成、已去背

## 背景
線上/公開版使用 `bg_garden.png` —— 一張 AI 生成的戴夫風**無人庭院**，貓咪會在牆頭、木箱、窗台、地面、花盆旁以各種姿勢出現。**不含任何真人影像**。
